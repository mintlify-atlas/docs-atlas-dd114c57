## RestClient Interface

The `RestClient` interface provides methods for interacting with Lavalink's REST API endpoints.

<Note>
  You typically access the REST client through a `Node` instance via `node.Rest()` rather than creating it directly.
</Note>

## HTTP Client

### Do

Executes a custom HTTP request with automatic authorization and host configuration.

<ParamField path="rq" type="*http.Request" required>
  The HTTP request to execute
</ParamField>

<ResponseField name="*http.Response" type="*http.Response">
  The HTTP response
</ResponseField>

<ResponseField name="error" type="error">
  Error if the request fails
</ResponseField>

```go
Do(rq *http.Request) (*http.Response, error)
```

<Note>
  This method automatically sets the Authorization header and replaces the host/scheme with the node's configuration.
</Note>

## Server Information

### Version

Retrieves the Lavalink server version.

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ResponseField name="string" type="string">
  The version string
</ResponseField>

<ResponseField name="error" type="error">
  Error if the request fails
</ResponseField>

```go
Version(ctx context.Context) (string, error)
```

**Example:**
```go
version, err := node.Rest().Version(ctx)
fmt.Printf("Lavalink version: %s\n", version)
```

### Info

Retrieves detailed server information.

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ResponseField name="*lavalink.Info" type="*lavalink.Info">
  Server info including version, build time, Git info, JVM info, plugins, and source managers
</ResponseField>

<ResponseField name="error" type="error">
  Error if the request fails
</ResponseField>

```go
Info(ctx context.Context) (*lavalink.Info, error)
```

**Example:**
```go
info, err := node.Rest().Info(ctx)
if err == nil {
    fmt.Printf("Version: %s\n", info.Version.Semver)
    fmt.Printf("Plugins: %v\n", info.Plugins)
}
```

<ResponseField name="error" type="error">
  Error if the request fails
</ResponseField>

```go
Stats(ctx context.Context) (*lavalink.Stats, error)
```

</ResponseField>

## Track Loading and Decoding

### LoadTracks

Loads tracks from an identifier.

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="identifier" type="string" required>
  URL or search query (e.g., "ytsearch:song name")
</ParamField>

<ResponseField name="*lavalink.LoadResult" type="*lavalink.LoadResult">
  Load result with tracks, playlist, or error
</ResponseField>

<ResponseField name="error" type="error">
  Error if the request fails
</ResponseField>

```go
LoadTracks(ctx context.Context, identifier string) (*lavalink.LoadResult, error)
```

**Example:**
```go
result, err := node.Rest().LoadTracks(ctx, "ytsearch:never gonna give you up")
if err == nil {
    switch data := result.Data.(type) {
    case lavalink.Track:
        fmt.Printf("Loaded track: %s\n", data.Info.Title)
    case lavalink.Playlist:
        fmt.Printf("Loaded playlist: %s\n", data.Info.Name)
    case lavalink.Search:
        fmt.Printf("Search results: %d tracks\n", len(data))
    }
}
```

## REST Endpoints

The following endpoint constants are available:

```go
EndpointBase          = "/v4"
EndpointVersion       = "/version"
EndpointInfo          = "/v4/info"
EndpointStats         = "/v4/stats"
EndpointUpdateSession = "/v4/sessions/%s"
EndpointPlayers       = "/v4/sessions/%s/players"
EndpointPlayer        = "/v4/sessions/%s/players/%s"
EndpointUpdatePlayer  = "/v4/sessions/%s/players/%s?noReplace=%t"
EndpointDestroyPlayer = "/v4/sessions/%s/players/%s"
EndpointLoadTracks    = "/v4/loadtracks?identifier=%s"
EndpointDecodeTrack   = "/v4/decodetrack?track=%s"
EndpointDecodeTracks  = "/v4/decodetracks"
EndpointWebSocket     = "/v4/websocket"
```

## Error Handling

All methods return `lavalink.Error` when the server responds with an error status:

```go
result, err := node.Rest().LoadTracks(ctx, "invalid://url")
if err != nil {
    if lavalinkErr, ok := err.(lavalink.Error); ok {
        fmt.Printf("Lavalink error: %s (severity: %s)\n", 
            lavalinkErr.Message, lavalinkErr.Severity)
    }
}
```
