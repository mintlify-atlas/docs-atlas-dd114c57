## Node Interface

The `Node` interface represents a connection to a Lavalink server and provides methods for managing tracks, sessions, and stats.

## NodeConfig Struct

Configuration for connecting to a Lavalink node.

<ParamField path="Name" type="string" required>
  Unique identifier for the node
</ParamField>

<ParamField path="Address" type="string" required>
  Host and port of the Lavalink server (e.g., "localhost:2333")
</ParamField>

<ParamField path="Password" type="string" required>
  Password for authenticating with the Lavalink server
</ParamField>

<ParamField path="Secure" type="bool">
  Whether to use secure connections (https/wss). Default: false
</ParamField>

<ParamField path="SessionID" type="string">
  Optional session ID for resuming previous sessions
</ParamField>

**Example:**
```go
config := disgolink.NodeConfig{
    Name:     "primary",
    Address:  "lavalink.example.com:443",
    Password: "youshallnotpass",
    Secure:   true,
}
```

## Status Constants

Node connection status values:

- `StatusConnecting` - Currently establishing connection
- `StatusConnected` - Successfully connected
- `StatusReconnecting` - Attempting to reconnect
- `StatusDisconnected` - Not connected

```go
type Status string

const (
    StatusConnecting   Status = "CONNECTING"
    StatusConnected    Status = "CONNECTED"
    StatusReconnecting Status = "RECONNECTING"
    StatusDisconnected Status = "DISCONNECTED"
)
```

## Track Loading

### LoadTracks

Loads tracks from an identifier (URL or search query).

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="identifier" type="string" required>
  URL or search query (e.g., "ytsearch:Never Gonna Give You Up")
</ParamField>

<ResponseField name="LoadResult" type="*lavalink.LoadResult">
  Load result containing tracks, playlists, or errors
</ResponseField>

<ResponseField name="error" type="error">
  Error if the request fails
</ResponseField>

```go
LoadTracks(ctx context.Context, identifier string) (*lavalink.LoadResult, error)
```

### LoadTracksHandler

Loads tracks and invokes callbacks based on the result type.

<ParamField path="ctx" type="context.Context" required>
  Context for the request
</ParamField>

<ParamField path="identifier" type="string" required>
  URL or search query
</ParamField>

<ParamField path="handler" type="AudioLoadResultHandler" required>
  Handler with callbacks for different result types
</ParamField>

```go
LoadTracksHandler(ctx context.Context, identifier string, handler AudioLoadResultHandler)
```

**Example:**
```go
handler := disgolink.NewResultHandler(
    func(track lavalink.Track) {
        fmt.Printf("Loaded track: %s\n", track.Info.Title)
    },
    func(playlist lavalink.Playlist) {
        fmt.Printf("Loaded playlist: %s\n", playlist.Info.Name)
    },
    func(tracks []lavalink.Track) {
        fmt.Printf("Search returned %d tracks\n", len(tracks))
    },
    func() {
        fmt.Println("No matches found")
    },
    func(err error) {
        fmt.Printf("Load failed: %v\n", err)
    },
)
node.LoadTracksHandler(ctx, "ytsearch:Never Gonna Give You Up", handler)
```

## Track Decoding

  Context for the request
</ParamField>

<ParamField path="encodedTrack" type="string" required>
  Base64-encoded track data
</ParamField>

<ResponseField name="Track" type="*lavalink.Track">
  The decoded track
</ResponseField>

<ResponseField name="error" type="error">
  Error if decoding fails
</ResponseField>

```go
DecodeTrack(ctx context.Context, encodedTrack string) (*lavalink.Track, error)
```

<ResponseField name="error" type="error">

## Connection Management

### Open

Opens the WebSocket connection to the Lavalink server.

<ParamField path="ctx" type="context.Context" required>
  Context for the connection attempt
</ParamField>

<ResponseField name="error" type="error">
  Error if connection fails
</ResponseField>

```go
Open(ctx context.Context) error
```

<Note>
  This method is automatically called by `Client.AddNode()`. You typically don't need to call it manually.
</Note>

### Close

Closes the WebSocket connection.

```go
Close()
```

## Errors

```go
var ErrNodeAlreadyConnected = errors.New("node already connected")
```

Returned when attempting to open a node that is already connected.
